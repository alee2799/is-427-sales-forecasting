Group Project: Store Sales - Time Series Forecasting
Description:
This program analyzes data like historic sales, number of transactions, and types of products sold
from a chain of Favorita supermarket stores. Methods like K-Means clustering, logistic regression, 
and SARIMAX are used to analyze the data and build a predictive model for future store sales on 
specific product families.

Requirements:
Python 3.x
Libraries: pandas, numpy, matplotlib.pyplot, seaborn
Imported from sklearn: train_test_split, LogisticRegression, classification_report, KMeans,
silhouette_score
Imported from statsmodel: SARIMAX, adfuller
    
How to run:
unzip folder, run the main file 'SALES_FORECAST.py'
